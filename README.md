# Safe EQ Real-Time Control System

Safe EQ is an embedded audio control system prototype designed to monitor audio input, evaluate signal conditions, and apply automated control behavior for safer listening. The project combines embedded hardware, sensor integration, Python control logic, and hardware/software validation across multiple platforms.

## Project Overview

This system was developed as a senior electrical engineering project at the University of California, Riverside. The prototype used MEMS microphones, microcontroller-based sensing, and external processing hardware to support real-time monitoring, threshold detection, and dynamic gain control.

This repository is a public portfolio version of the project and is intended to show the embedded hardware, control logic, testing, and system integration work behind the prototype.

## Hardware Used

* ESP32-S3
* Raspberry Pi 4
* NVIDIA Jetson Orin Nano
* ICS-43434 MEMS microphones
* Custom 3D-printed microphone enclosure
* Bench test equipment including oscilloscopes, and digital multimeters.

## Communication Interfaces

* I2S
* I2C
* UART

## Software / Tools

* Python
* C/C++
* Embedded firmware development
* MATLAB for analysis and documentation
* Serial communication debugging

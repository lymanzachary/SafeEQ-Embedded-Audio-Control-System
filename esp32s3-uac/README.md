# ESP32 Stereo MEMS Microphone USB Audio Interface

This folder contains the ESP32 code I wrote for the Safe EQ project. Its purpose was to use the ESP32 as a USB audio interface for stereo MEMS microphones, allowing microphone audio to be captured and passed into the rest of the Safe EQ system for processing and testing.

## Purpose

The ESP32 firmware was developed to handle stereo microphone input and provide a usable audio interface for the project. This helped support real-time audio capture, system testing, and hardware/software integration during prototype development.

## Main Functions

* Captures audio from stereo MEMS microphones
* Interfaces with digital microphone hardware
* Supports USB audio behavior for system integration
* Provides audio input for downstream processing and validation
* Helps test microphone behavior, signal quality, and system responsiveness

## Project Context

This code was part of Safe EQ, a senior design project focused on real-time audio monitoring and control. My work on this portion focused on the embedded audio capture side of the system, including microphone integration, firmware development, and debugging.

## Hardware Context

* ESP32-based development board
* Stereo MEMS microphones
* USB connection to host system

## Notes

This folder is included as part of a public portfolio version of the Safe EQ project. The code reflects my work on the ESP32 microphone interface portion of the system and may not include every experimental or development file from the original project.
